docker rmi <image_name> or <image_id>
    Deletes image

docker rm <container_name> or <container_id>
    Deletes container

we cannot delete image if it has a container created from it
First we need to delete the container and then that image

LIST ALL IMAGES

C:\Users>docker images
                                                                                                          i Info →   U  In Use
IMAGE                ID             DISK USAGE   CONTENT SIZE   EXTRA
hello-world:latest   f7931603f70e       20.3kB         3.96kB    U
ubuntu:latest        c35e29c94501        117MB         29.7MB    U

TRY TO DELETE IMAGE WITH CONTAINER CREATED

C:\Users>docker rmi ubuntu:latest
Error response from daemon: conflict: unable to delete ubuntu:latest (must be forced) - container 51ea8fb8c6e6 is using its referenced image c35e29c94501

LIST ALL CONTAINERS AND CHECK WHICH ONE USES THAT IMAGE

C:\Users>docker ps -a
CONTAINER ID   IMAGE                COMMAND       CREATED             STATUS                     PORTS     NAMES
51ea8fb8c6e6   ubuntu               "/bin/bash"   36 minutes ago      Exited (0) 5 minutes ago             nifty_satoshi
d3dfa042322c   hello-world:latest   "/hello"      About an hour ago   Exited (0) 5 minutes ago             quizzical_jang

REMOVE THAT CONTAINER WHICH USES IMAGE

C:\Users>docker rm nifty_satoshi
nifty_satoshi

CHECK CONTAINER DELETE

C:\Users>docker ps -a
CONTAINER ID   IMAGE                COMMAND    CREATED             STATUS                     PORTS     NAMES
d3dfa042322c   hello-world:latest   "/hello"   About an hour ago   Exited (0) 5 minutes ago             quizzical_jang

NOW DELETE THAT IMAGE

C:\Users>docker rmi ubuntu:latest
Untagged: ubuntu:latest
Deleted: sha256:c35e29c9450151419d9448b0fd75374fec4fff364a27f176fb458d472dfc9e54

CHECK IMAGE DELETE

C:\Users>docker images
                                                                                                          i Info →   U  In Use
IMAGE                ID             DISK USAGE   CONTENT SIZE   EXTRA
hello-world:latest   f7931603f70e       20.3kB         3.96kB    U
