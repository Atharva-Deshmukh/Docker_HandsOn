docker --version
    To check if docker is installed: 

docker (and then press enter)
    To get a list of all commands: 

docker pull <image_name>
    This command allows you to pull any image which is present in the official 
    registry of docker, Docker hub. By default, it pulls the latest image, 
    but you can also mention the version of the image. By default, :latest tag is added

docker images
    To list all the images in local

docker pull <imageName>
docker pull hello-world
    To download a specific image (with default tag = :latest)
